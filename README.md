# eQuran

eQuran adalah proyek berbasis PHP yang dirancang untuk menyediakan fungsionalitas Al-Quran melalui shortcode.

## Deskripsi

Proyek ini bertujuan untuk mempermudah integrasi ayat-ayat Al-Quran, terjemahan, dan audio ke dalam aplikasi web menggunakan shortcode sederhana. Dibuat agar ringan dan mudah digunakan.

## Fitur

- Dukungan shortcode untuk menampilkan ayat.
- Opsi tampilan yang dapat disesuaikan.
- Ringan dan efisien.

## Instalasi

1. Clone repositori:
   ```bash
   git clone https://github.com/bungrahman/equran.git
   ```

2. Sertakan `shortcode_function.php` dalam file `functions.php` proyek atau tema Anda.

## Penggunaan

Gunakan shortcode yang disediakan dalam konten Anda untuk menampilkan konten Al-Quran.

Contoh:
```php
[equran_verse surah="1" ayah="1"]
```
